# FoClASS

* concepts==0.7.10
* graphviz==0.6
* matplotlib==2.0.0
* numpy==1.12.1
* pandas==0.19.2
* pyparsing==2.2.0
* python-dateutil==2.6.0
* pytz==2017.2
* requests==2.13.0
* scipy==0.19.0
* six==1.10.0
* wrapt==1.10.10

# Usage

```
./ginsimToInputFile.sh -d|--directory [Directory] -a|--archive [GINsim Archive] -o|--option [Options]


./analysisFCA.sh -d [Directory] -s [Signature File]
```
